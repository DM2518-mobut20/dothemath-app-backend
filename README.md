# dothemath-app-backend

## Installation

`git clone ...`

`npm install`

## Köra lokalt

Följande miljövariabler behöver sättas

- SLACK_BOT_TOKEN = värdet finns i slack-chatten
- SLACK_SIGNING_SECRET = värdet finns i slack-chatten
- PORT = 80 (eller ska det vara 443?)

Kör sedan

`npm run watch-debug`
